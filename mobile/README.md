## 🆘 Sobre

O <strong>Be The Hero</strong> é uma plataforma para ongs possa achar heróis que possam ajudá-los nas causas que elas têm.
Projeto criado na <strong>Semana OmniStack 11</strong>.

  <li>Expo</li>
